dynsim

An R implementation of dynamic simulations of autoregressive relationships.

Laron K Williams, Guy D Whitten, and Christopher Gandrud

About

The dynsim package underdevelopment implements Williams and Whitten's (2011, 2012) method for dynamic simulations of autoregressive relationships in R.

Installation

dynsim is available on CRAN

You can also easily install the latest development version with the devtools package:

devtools::install_github("dynsim", "christophergandrud")
